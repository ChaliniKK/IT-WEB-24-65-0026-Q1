# 🚀 Taskia – Task Management Landing Page

## 🌟 Project Overview

**Taskia** is a modern, fully responsive task management landing page built using **HTML5, Tailwind CSS, and Vanilla JavaScript**.

This project focuses on clean UI design, responsive layouts, and interactive client-side functionality while maintaining a professional and minimal aesthetic.

It demonstrates strong front-end development skills including layout structuring, responsive styling, and DOM manipulation.

---

## ✨ Key Features

- ✅ Fully Responsive Design (Mobile-First Approach)
- ✅ Modern UI built with Tailwind CSS
- ✅ Interactive Email Input Handling
- ✅ Button Click Animations
- ✅ Hover Scale Effects
- ✅ Sticky Navigation Bar
- ✅ Optimized Typography (Google Fonts – Inter)
- ✅ Clean and Structured Codebase

---

## 🛠️ Technologies Used

- **HTML5** – Page Structure  
- **Tailwind CSS (CDN)** – Styling & Responsiveness  
- **JavaScript (Vanilla JS)** – Client-Side Interactivity  
- **Google Fonts** – Typography Enhancement

## 💡 Core Functionality

### 📧 Email Input Handling

- Detects **Enter key submission**
- Displays a confirmation alert
- Automatically clears the input field

### 🎯 Button Interaction Effects

- Adds a subtle **pulse animation**
- Enhances user interaction feedback
